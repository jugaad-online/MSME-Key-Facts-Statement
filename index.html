Loan Inputs

Bulk Import (CSV) — multiple loans

Download the template (it ships with two example rows — an ECLGS 5.0 and an MSME Term Loan), fill one row per loan, then import to generate the full KFS (Annexure-A Parts 1&2, Annexure-A(1) & Annexure-A(2)) for every loan at once. Each row uses the exact values you enter (the product auto-fill applies only in the form). Every row is validated on import; rows with invalid/missing data are listed and skipped.
